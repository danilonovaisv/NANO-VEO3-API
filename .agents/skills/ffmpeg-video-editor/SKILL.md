# 🎞️ FFmpeg Video Editor — CLI Media Processing Skill

## Overview

FFmpeg is the workhorse for low-level media manipulation. Use this skill for operations that don't require programmatic React composition (Remotion) — concatenation, audio mixing, subtitle burning, format conversion, and quality optimization.

## When to Use

- Concatenating multiple AI-generated clips into a single video
- Adding audio tracks to silent video clips from Veo3/Sora
- Burning subtitles (SRT) into final renders
- Converting formats and optimizing for web delivery
- Extracting frames for quality review
- Trimming or cutting clips to exact timestamps

## Safety Rules

- **Always use `set -euo pipefail`** in shell scripts
- **Never overwrite** files in `public/out/` without explicit user approval
- **Always verify** input files exist before processing
- **Use `ffprobe`** to validate output metadata after rendering

## Common Recipes

### 1. Add Audio to Silent Video

```bash
#!/usr/bin/env bash
set -euo pipefail

VIDEO="${1:?Usage: add-audio.sh <video.mp4> <audio.mp3>}"
AUDIO="${2:?Missing audio file}"
OUTPUT="public/out/$(basename "$VIDEO" .mp4)_with_audio.mp4"

ffmpeg -i "$VIDEO" -i "$AUDIO" \
  -c:v copy \
  -c:a aac -b:a 192k \
  -shortest \
  "$OUTPUT"

echo "✅ Output: $OUTPUT"
ffprobe -v quiet -print_format json -show_format "$OUTPUT" | head -20
```

### 2. Concatenate Multiple Clips

```bash
#!/usr/bin/env bash
set -euo pipefail

# Create file list
FILELIST=$(mktemp /tmp/ffmpeg_concat_XXXXXX.txt)
for f in public/raw/clip_*.mp4; do
  echo "file '$(realpath "$f")'" >> "$FILELIST"
done

ffmpeg -f concat -safe 0 -i "$FILELIST" \
  -c:v libx264 -preset slow -crf 18 \
  -c:a aac -b:a 192k \
  "public/out/concatenated_$(date +%Y%m%d_%H%M%S).mp4"

rm "$FILELIST"
echo "✅ Concatenation complete"
```

### 3. Burn Subtitles (SRT)

```bash
#!/usr/bin/env bash
set -euo pipefail

VIDEO="${1:?Usage: burn-subs.sh <video.mp4> <subtitles.srt>}"
SUBS="${2:?Missing subtitle file}"
OUTPUT="public/out/$(basename "$VIDEO" .mp4)_subtitled.mp4"

ffmpeg -i "$VIDEO" -vf "subtitles=${SUBS}:force_style='\
  FontName=Inter,\
  FontSize=24,\
  PrimaryColour=&HFFFFFF&,\
  OutlineColour=&H000000&,\
  Outline=2,\
  Shadow=1,\
  Alignment=2,\
  MarginV=50'" \
  -c:a copy \
  "$OUTPUT"

echo "✅ Subtitled: $OUTPUT"
```

### 4. Extract Key Frames for Review

```bash
#!/usr/bin/env bash
set -euo pipefail

VIDEO="${1:?Usage: extract-frames.sh <video.mp4>}"
OUTDIR="public/raw/frames_$(basename "$VIDEO" .mp4)"
mkdir -p "$OUTDIR"

# Extract 1 frame per second
ffmpeg -i "$VIDEO" -vf "fps=1" "$OUTDIR/frame_%04d.png"

echo "✅ Frames extracted to: $OUTDIR"
ls -la "$OUTDIR" | head -10
```

### 5. Convert to Web-Optimized H.264

```bash
#!/usr/bin/env bash
set -euo pipefail

INPUT="${1:?Usage: optimize.sh <input.mp4>}"
OUTPUT="public/out/$(basename "$INPUT" .mp4)_web.mp4"

ffmpeg -i "$INPUT" \
  -c:v libx264 -preset slow -crf 18 \
  -c:a aac -b:a 192k \
  -movflags +faststart \
  -pix_fmt yuv420p \
  "$OUTPUT"

echo "✅ Web-optimized: $OUTPUT"
```

### 6. Add Crossfade Transitions

```bash
#!/usr/bin/env bash
set -euo pipefail

CLIP1="${1:?Usage: crossfade.sh <clip1.mp4> <clip2.mp4>}"
CLIP2="${2:?Missing second clip}"
FADE_DURATION=1  # seconds
OUTPUT="public/out/crossfade_output.mp4"

# Get duration of first clip
DUR1=$(ffprobe -v error -show_entries format=duration -of default=nw=1:nk=1 "$CLIP1")
OFFSET=$(echo "$DUR1 - $FADE_DURATION" | bc)

ffmpeg -i "$CLIP1" -i "$CLIP2" \
  -filter_complex "[0:v][1:v]xfade=transition=fade:duration=${FADE_DURATION}:offset=${OFFSET}[v]; \
                    [0:a][1:a]acrossfade=d=${FADE_DURATION}[a]" \
  -map "[v]" -map "[a]" \
  "$OUTPUT"

echo "✅ Crossfade: $OUTPUT"
```

## Validation Command

Always run after rendering:

```bash
ffprobe -v quiet -print_format json -show_streams -show_format "$OUTPUT" | \
  jq '{duration: .format.duration, size: .format.size,
       video_codec: .streams[0].codec_name, resolution: "\(.streams[0].width)x\(.streams[0].height)",
       audio_codec: .streams[1].codec_name}'
```
